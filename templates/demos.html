Gauge Types

Gauge directive lets you create three distinct types of gauges:

  • full (default) - a circular gauge
  • semi - a semi-circular gauge
  • arch - arc like gauge less than a full circle.

Gauge Styles

Gauge directive provides two cap styles for Gauge lines:

  • butt (default) - gives a flat edge to gauge line
  • round - gives a rounded edge to gauge line

Gauge Thickness

Thickness of gauge can be set using `thick` attribute. It takes a positive integer value.

Dynamic Gauge Value

The gauge value can be specified directly or bound to a scope variable defined in the controller. The gauge will udpate the display when the value is updated in the scope. Click the button to see its working.

Gauge Themes

Themes for gauge widget can be set manually using foreground color and background color attributes. In upcoming releases, it will contain set of named themes which can be applied based on choice.

  • foreground-color - #FFCC66 (default)
  • background-color - #EEE (default)

It can accept any valid CSS color value.
